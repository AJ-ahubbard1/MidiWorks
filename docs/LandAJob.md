## INTERVIEW PREP - KEY QUESTIONS

### Technical Questions About MidiWorks

**"Walk me through your MidiWorks architecture"**
- Model-View separation (AppModel vs Panels)
- Command pattern for undo/redo
- Real-time constraints (10ms timer, MIDI callback)
- Multi-threaded architecture (RtMidi callback + main loop)

**"How do you handle real-time timing requirements?"**
- 10ms update timer in MainFrame
- RtMidi callback on separate thread
- Thread-safe queue for MIDI input
- State management in main thread

**"Tell me about a hard bug you debugged"**
- Bug #8: Loop recording active note tracking
- Complex state with mActiveNotes, mRecordingBuffer, mRecordingBufferIterator
- Edge cases at loop boundaries
- Learned: Draw state machines before coding complex state

**"How does this relate to embedded systems?"**
- Hardware integration (MIDI devices)
- Timing constraints (real-time processing)
- Cross-platform Linux builds (CMake)
- Resource constraints (low latency, efficient code)
