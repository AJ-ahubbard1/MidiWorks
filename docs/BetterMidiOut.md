Recommended: FluidSynth + SoundFonts

FluidSynth is the go-to open-source software synthesizer:
- Cross-platform (Windows, Linux, macOS)
		- Uses SoundFont 2 (.sf2) format
		- Actively maintained, very stable
		- Can be integrated into apps or used standalone

		Windows: VirtualMIDISynth (Easiest)

		https://coolsoft.altervista.org/en/virtualmidisynth
		- Based on FluidSynth
- Installs as a Windows MIDI device (shows up in your device list automatically)
		- GUI for managing SoundFonts
		- Just install it, load a SoundFont, and select it in MidiWorks!

		Quick Setup

1. Install VirtualMIDISynth (Windows) or FluidSynth (cross-platform)
		2. Download GeneralUser GS SoundFont
		3. Load it in VirtualMIDISynth settings
		4. In MidiWorks, select "VirtualMIDISynth #1" as output device
